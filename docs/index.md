# Projektbeschreibung

## Projektname
HTL INN - 4. Klasse - CC - Git

## Beschreibung
Dieses Projekt dient als Dokumentation und Übung für den Umgang mit Git in der 4. Klasse der HTL INN. Es enthält verschiedene Dateien und Anleitungen, um die Grundlagen der Versionskontrolle zu erlernen und anzuwenden.

## Inhalte
- **Dokumentation**: Enthält Anleitungen und Erklärungen zu Git-Befehlen.
- **Beispiele**: Praktische Beispiele zur Anwendung von Git.
- **Übungen**: Aufgaben zur Festigung des Wissens.

## Ziel
Das Ziel dieses Projekts ist es, die Schüler*innen mit den grundlegenden Konzepten und der praktischen Anwendung von Git vertraut zu machen.
